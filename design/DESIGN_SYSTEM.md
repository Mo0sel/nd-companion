# N&D Companion Design System

## Status

Approved

## Philosophy

N&D Companion is a productivity tool for Dungeon Masters.

The interface should feel closer to VS Code, Linear and Figma than to a fantasy RPG interface.

The goal is to reduce cognitive load during session preparation and live play.

---

# Layout Principles

The overall layout is considered stable.

Do not redesign it.

Future features should fit naturally into the existing interface.

Major layout changes require explicit approval.

---

# Information Hierarchy

1. Campaign Context

2. Workspace Navigation

3. Current Beat

4. Companion Memory

5. Session Notes

Information higher in the list should receive greater visual emphasis.
