# N&D Companion Design

This folder contains the visual source of truth for the N&D Companion.

## Structure

figma-export/

- Screens exported from Figma.

- Mockups used during implementation.

DESIGN_[SYSTEM.md](http://SYSTEM.md)

- Design principles.

- Layout rules.

- Component behavior.

- UX conventions.

DESIGN_[TOKENS.md](http://TOKENS.md)

- Colors.

- Typography.

- Spacing.

- Border radius.

- Shadows.

- CSS variables used by the implementation.

The approved Figma layout is considered the baseline for the product.

Future work should extend the interface rather than redesign it.
