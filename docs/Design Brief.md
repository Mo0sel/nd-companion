# N&D Companion

Version: 0.2.7

## Vision

N&D Companion is a professional campaign companion for Foundry VTT.

It is not a campaign manager.

It is not a wiki.

It is not a note-taking application.

Its purpose is to reduce Dungeon Master cognitive load before and during live sessions.

The Companion should feel closer to tools like Linear, Notion, Arc Browser and Figma than traditional tabletop software.

The focus is clarity, speed and storytelling.

# Workspace Responsibilities

## CAMPAIGN

Campaign source of truth.

Contains:

- Main Quests

- Side Quests

- Companion Quests

Each Quest contains:

- Overview

- Entries

- References

Campaign writing happens here.

---

## PREPARE

Session builder.

Imports Quest Entries.

Allows session-specific preparation.

Nothing here should overwrite Campaign content.

---

## PLAY

Live dashboard.

Runs the prepared session.

Shows Current Entry.

Speech Notes.

Objectives.

XP.

Notes.

Twists.

Possible Outcomes.

Session Notes.

Everything should be optimized for running a live game.

---

## ACTORS

Campaign memory.

Contains:

Companion Memory

Future Reputation graph.

# Design Principles

Less scrolling.

More scanning.

Everything important visible.

Cards over documents.

Typography over decoration.

Hierarchy over borders.

Dark mode first.

Fast under pressure.

No duplicated information.

Campaign is the source of truth.

---

# Information Hierarchy

1. Workspace navigation — neutral.
2. Page context — subtle accent.
3. Primary object — largest and highest emphasis.
4. Section group — muted color with a subtle divider.
5. Card title — consistent uppercase micro-label.

Use hierarchy before adding color or decoration.
